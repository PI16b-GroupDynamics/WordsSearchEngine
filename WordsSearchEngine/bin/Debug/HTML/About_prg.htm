Информация о программе

В программе предусмотрены 5 форм. Ниже приведена инструкция и описание к каждой из них.

1. Форма "Приветствие"

На этой форме располагаются фото авторов проекта и краткое описание проекта.

2. Главная форма проекта

На этой форме располагается 80% функционала всей программы: поиск в тексте или поиск в файлах

по заданным критериям.

3. Форма параметров.

На этой форме находятся все главные устанавливаемые пользователем параметры приложения.

4. Форма авторизации.

Эта форма предназначена для авторизации пользователя. Также возможна регистрация.

5. Форма регистрации.

Эта форма позволяет новому пользователю зарегистрироваться.
