Инструкция пользователю

На главной форме представлены критерии поиска слов в тексте. Прежде, Вам стоит выбрать, с помощью флаговых кнопок,

хотите Вы осуществлять поиск в файлах или по тексту. После выберите критерии, по которым будет осуществляться поиск.

Если поиск осуществляется по тексту, Вы можете указать его название. Нажмите кнопку "Начать", и дождитесь обработки текста.

При желании, Вы можете выгрузить текст из файла с помощью иконки в виде стрелочки, а также, по окончанию работы

Вы можете сохранить результат поиска слов нажатием на иконку в виде дискеты.

Чтобы воспользоваться поиском в файлах необхоимо и достаточно только нажать кнопку "Обзор..." для выбора директории,

в которой будет осуществляться поиск по выбранным критериям.

В контекстном меню присутствуют 4 раздела, а именно, "Файл" - осуществление выгрузки и сохранения исходного текста,

"Настройки" - параметры сохранения результирующего файла, "Справка" - инструкция пользователю и раздел о программе и

"Выход" - осуществление выхода из приложения.

С помощью кнопки "Вход" Вы можете авторизоваться или зарегистрироваться.
